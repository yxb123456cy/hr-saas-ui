# HR SAAS系统需求规划与界面清单

# 一、项目核心定位与目标

本HR SAAS系统聚焦企业“内部人事管理”与“自主招聘”两大核心场景，通过前后端分离架构，实现不依赖第三方招聘平台的全流程招聘管理，同时满足HR与普通员工的内部人事协作需求，支持多角色权限精细化管控，提升企业人事管理效率与招聘转化效果。

# 二、角色及核心权限需求

## 2.1 HR总管

系统最高权限角色，负责系统全局配置、角色权限管理、数据统计分析，同时具备普通HR的全部权限。核心权限包括：

- 系统配置：基础参数设置（如部门结构、职位级别、薪酬体系）、招聘流程配置、通知模板配置等；

- 权限管理：角色创建/编辑/删除、权限分配、用户账号管理（新增/禁用/重置密码）；

- 数据管理：全公司人事数据查看、招聘数据统计、报表导出、数据备份与恢复；

- 全流程审批：招聘需求审批、录用审批、离职审批等所有关键流程的最终审批；

- 全局操作：覆盖普通HR与普通员工的全部操作权限。

## 2.2 普通HR

兼具普通员工权限，核心负责招聘全流程执行与内部人事基础管理。核心权限包括：

- 招聘管理：职位发布/编辑/下架、简历筛选/查看/归档、面试安排/通知、录用offer发放；

- 人事管理：员工信息录入/维护、合同签订/续签/解除、考勤数据统计、薪酬核算辅助；

- 流程发起：招聘需求申请、员工入职/转正/调岗/离职流程发起、审批跟进；

- 员工权限：个人信息查看/修改、考勤打卡、请假/加班申请（同普通员工）。

## 2.3 公司普通员工

核心关注个人人事相关操作与信息查看。核心权限包括：

- 个人中心：个人基本信息查看/修改、入职资料补充、劳动合同查看；

- 考勤管理：日常打卡、请假/加班/出差申请、考勤记录查看；

- 审批相关：参与涉及自身的入职/转正/调岗/离职流程协作、查看审批进度；

- 其他：公司公告查看、部门成员信息查看（权限范围内）。

## 2.4 求职者

外部角色，核心用于浏览职位、投递简历、查看应聘进度。核心权限包括：

- 职位浏览：查看企业发布的公开职位信息、筛选职位（按部门、薪资、经验要求等）；

- 简历管理：注册/登录账号、创建/编辑/上传个人简历、查看简历状态；

- 应聘操作：投递职位、查看应聘进度（已投递、初筛、面试、录用/淘汰）、接收面试/录用通知。

# 三、核心功能需求拆解

## 3.1 系统基础模块

支撑系统正常运行的基础功能，覆盖所有角色的通用操作。

- 用户认证：登录（账号密码/验证码）、注册（求职者专属）、注销、密码重置（短信/邮箱）、记住密码、单点登录支持；

- 权限控制：基于Satoken的角色权限校验、接口权限拦截、菜单权限动态渲染；

- 通知中心：系统消息推送（审批通知、面试通知等）、消息列表查看、已读/未读标记、消息删除；

- 系统配置：HR总管专属的全局参数配置、部门结构管理、职位体系管理、字典管理（如学历、专业、职称等）。

## 3.2 招聘管理模块（自主招聘场景）

实现企业不依赖第三方平台的全流程招聘，覆盖从职位发布到员工录用的全环节。

- 招聘需求管理：普通HR发起招聘需求申请（含职位名称、招聘人数、薪资范围、任职要求等）、HR总管审批；

- 职位管理：审批通过后发布职位、编辑职位信息、设置职位是否公开、下架过期职位、职位预览；

- 简历管理：求职者自主投递简历、HR手动上传简历、简历智能筛选（按关键词、条件匹配）、简历分类归档（待筛选、初筛通过、淘汰、面试中）、简历详情查看；

- 面试管理：HR创建面试计划（面试时间、地点、面试官）、发送面试通知（短信/邮箱）、面试官录入面试评价、面试结果同步（通过/淘汰/复试）；

- 录用管理：HR发起录用申请、HR总管审批、生成录用offer（模板化）、发送offer给求职者、跟踪offer接受状态、录用资料收集提醒。

## 3.3 内部人事管理模块

覆盖企业内部员工从入职到离职的全生命周期管理，满足HR与普通员工的协作需求。

- 员工信息管理：HR录入/维护员工基础信息（个人信息、联系方式、教育经历、工作经历等）、员工自主补充/修改个人信息、员工信息批量导入/导出；

- 入职管理：新员工入职流程发起、入职资料审核、入职引导（公司制度、部门介绍）、试用期管理（试用期提醒、转正申请）；

- 合同管理：劳动合同签订、续签提醒、合同到期预警、解除/终止合同办理、合同档案管理；

- 考勤管理：考勤规则配置（上下班时间、打卡规则、加班规则）、员工日常打卡（正常打卡/补卡申请）、考勤数据统计（迟到、早退、旷工、加班时长）、考勤报表生成；

- 假期管理：请假类型配置（事假、病假、年假、婚假等）、员工请假/加班/出差申请、审批流程、假期余额查询；

- 调岗调薪管理：调岗申请、调薪申请、审批流程、岗位/薪资信息更新；

- 离职管理：员工主动离职申请、公司辞退流程、工作交接记录、离职证明开具、离职原因统计。

## 3.4 数据统计与报表模块

为HR总管和普通HR提供数据决策支持，覆盖招聘与人事核心数据。

- 招聘数据统计：职位投递量、初筛通过率、面试通过率、录用转化率、各职位招聘进度、招聘周期统计；

- 人事数据统计：员工总数、部门人员分布、学历/年龄/工龄分布、离职率统计、试用期转正率；

- 考勤数据报表：月度考勤汇总、加班时长统计、请假频次统计、异常考勤分析；

- 报表导出：支持Excel/PDF格式导出，自定义报表统计维度。

## 3.5 文档管理模块（基于Minio）

实现简历、合同、员工资料等文件的存储与管理，依托Minio实现文件的安全存储、访问控制。

- 文件上传：简历（求职者/HR上传）、劳动合同扫描件、员工证件照、入职资料等；

- 文件管理：文件分类归档、文件预览、文件下载、文件权限控制（如求职者只能查看自己的简历）；

- 文件安全：文件加密存储、访问日志记录、过期文件清理。

# 四、全部界面清单（按角色与模块划分）

## 4.1 通用公共界面

- 登录界面：支持账号密码登录、验证码登录、忘记密码入口（分员工/求职者）；

- 注册界面：仅求职者使用，含基本信息填写、手机号验证、密码设置；

- 密码重置界面：通过短信/邮箱验证重置密码；

- 通知中心界面：展示所有系统消息，支持已读/未读筛选、消息搜索、删除；

- 404/500错误界面：系统异常提示与返回首页入口；

- 权限不足提示界面：无权限操作时的提示页面。

## 4.2 HR总管专属界面

- 系统配置首页：全局配置概览，提供各配置模块入口；

- 部门管理界面：部门列表展示、新增/编辑/删除部门、部门排序；

- 角色权限管理界面：角色列表、新增/编辑角色、权限分配弹窗；

- 用户管理界面：全公司用户列表、新增/禁用用户、重置密码、用户信息编辑；

- 全局参数配置界面：职位级别、薪酬体系、招聘流程节点、通知模板等配置；

- 数据统计总览界面：整合招聘、人事、考勤核心数据仪表盘，支持数据筛选（时间范围等）；

- 数据报表管理界面：各类报表生成、预览、导出，自定义报表维度；

- 数据备份与恢复界面：备份记录查看、手动备份、恢复数据操作。

## 4.3 普通HR界面（含普通员工界面）

### 4.3.1 招聘管理相关界面

- 招聘需求申请界面：填写招聘需求信息（职位、人数、薪资等）、提交审批；

- 招聘需求审批列表界面：查看待审批/已审批/驳回的招聘需求；

- 职位管理界面：职位列表（已发布/草稿/已下架）、新增/编辑职位、职位预览/发布/下架；

- 简历管理界面：简历列表（按状态筛选）、简历详情查看、简历筛选（关键词/条件）、简历归档/删除；

- 面试管理界面：面试计划列表、新增面试计划、发送面试通知、录入面试评价、面试结果编辑；

- 录用管理界面：录用申请列表、发起录用申请、生成offer模板、发送offer、查看offer接受状态；

- 招聘数据统计界面：查看所负责职位的招聘进度、投递量、通过率等数据。

### 4.3.2 人事管理相关界面

- 员工信息管理界面：员工列表、新增/编辑员工信息、员工信息导入/导出、查看员工详情；

- 入职管理界面：新员工入职流程列表、发起入职流程、审核入职资料、查看入职进度；

- 合同管理界面：合同列表（按状态筛选：在用/到期/解除）、签订合同、续签提醒、合同详情查看、合同归档；

- 考勤配置界面：设置考勤规则、上下班时间、加班/补卡规则；

- 考勤统计界面：查看全公司/部门考勤数据、生成考勤报表、异常考勤处理；

- 假期管理界面：请假类型配置、查看员工假期余额、审批请假/加班/出差申请；

- 调岗调薪管理界面：调岗/调薪申请列表、审批申请、更新员工岗位/薪资信息；

- 离职管理界面：离职申请列表、审批离职申请、记录工作交接情况、开具离职证明。

### 4.3.3 普通员工基础界面

- 个人中心首页：展示个人基本信息、入职时间、岗位、部门等核心信息；

- 个人信息编辑界面：修改个人联系方式、紧急联系人、教育经历等信息；

- 考勤打卡界面：展示今日打卡状态、上下班打卡按钮、补卡申请入口、查看考勤记录；

- 假期申请界面：选择申请类型（请假/加班/出差）、填写申请信息、提交审批、查看申请进度；

- 个人合同查看界面：查看自己的劳动合同信息、签订时间、到期时间；

- 公司公告界面：查看公司/部门发布的公告、公告搜索、查看详情。

## 4.4 求职者专属界面

- 职位浏览首页：展示企业公开职位列表、支持按部门/薪资/经验/学历筛选、职位搜索；

- 职位详情界面：展示职位职责、任职要求、薪资范围、招聘人数、申请按钮；

- 个人简历管理界面：创建/编辑简历（基本信息、教育经历、工作经历、项目经验、技能证书等）、上传简历附件、预览简历；

- 我的应聘界面：展示已投递职位列表、应聘进度（已投递/初筛/面试/录用/淘汰）、查看面试通知/offer信息；

- 求职者个人中心：查看个人账号信息、修改密码、消息通知（面试/录用通知）。

# 五、界面关联与流程衔接说明

1. 招聘流程界面衔接：招聘需求申请 → 审批通过 → 职位发布 → 求职者投递简历 → 简历筛选 → 面试安排 → 面试评价 → 录用申请 → 审批通过 → 发放offer → 入职流程；

2. 人事生命周期界面衔接：入职申请 → 资料审核 → 试用期管理 → 转正申请 → 合同签订 → 日常考勤/假期管理 → 调岗调薪申请 → 离职申请 → 工作交接 → 离职证明开具；

3. 权限关联：不同角色登录后，系统根据角色权限动态渲染菜单与界面按钮（如求职者无法查看内部人事管理界面，普通员工无法查看全局数据统计界面）；

4. 文档关联：简历、合同等文件上传后，在对应详情界面（如简历详情、合同详情）提供预览/下载入口，依托Minio实现文件的安全访问。
